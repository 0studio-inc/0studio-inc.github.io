# About

optopuss is an open-source version of the pythOPT package for statistical evaluation of optimization methods. It is available as Docker image on AWS.

pythOPT is a software package that offers reproducible research for optimization methods. Examples of its application to theoretical and experimental cases are given in the thesis that is is a part of: pythOPT: A problem-solving environment for optimization methods (opens new window).
